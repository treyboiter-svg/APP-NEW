🎲 Odds & Market Fields
FieldTabSourceDescription & Formula
Away ML / Home MLMoneylineSportsbooksAmerican moneyline odds as scraped from each sportsbook. Raw value, no adjustment.
Away Raw% / Home Raw%MoneylineCalculatedImplied probability from American odds.
pos: 100/(odds+100)  |  neg: |odds|/(|odds|+100)
Away NV Prop% / Home NV Prop%MoneylineCalculatedNo-vig (vig-removed) proportional method. Raw implied probs normalised to sum to 100%.
nv_prop = raw_implied / (raw_home + raw_away) × 100
Vig%MoneylineCalculatedBookmaker overround. Sum of raw implied probs minus 100%.
vig = (raw_home + raw_away) − 100
Consensus MLMoneylineCalculatedAverage no-vig implied probability across all books with valid lines. Converted back to American.
Spread Point / MLSpreadSportsbooksPoint spread and price per book. Implied% same formula as moneyline.
O/U Point / MLTotal O/USportsbooksOver/Under total and price per book.
Kalshi Home Implied%Pred MarketsKalshiYes-contract last price × 100. Direct market-implied probability, no vig model applied.
Polymarket Home Implied%Pred MarketsPolymarketCLOB mid-price × 100. Best bid/ask midpoint from Polymarket order book.
Kalshi Disparity%DisparityCalculatedDifference between Kalshi market probability and consensus sportsbook no-vig probability.
disp = kalshi_implied − consensus_nv_prop
Poly Disparity%DisparityCalculatedSame formula using Polymarket implied instead of Kalshi.
Kalshi EV%Pred MarketsCalculatedExpected value if betting sportsbook price using Kalshi as “true” probability.
ev = kalshi_implied × (book_win_payout) − (1 − kalshi_implied)
Power Method %Power MethodCalculatedShin / power devig method. Solves for true probs via iterative root-finding on the power equation:
Σ(p_i^k) = 1  →  solve k, then p_true = p_raw^k / Σ(p_raw^k)
🌡 Weather & Physics Fields
FieldSourceAPI Path / Formula
Temperature (F/C)OpenWeatherforecast[i].main.temp (Kelvin) → converted. Nearest forecast hour to game time.
Relative Humidity%OpenWeatherforecast[i].main.humidity
Wind Speed (mph/kmh)OpenWeatherforecast[i].wind.speed (m/s) → ×2.237 for mph, ×3.6 for km/h
Wind Direction (deg)OpenWeatherforecast[i].wind.deg — meteorological convention: direction wind blows FROM.
Precipitation (in/h)OpenWeatherforecast[i].rain["3h"] / 3 (mm/h) → ÷25.4 for in/h
Station Pressure (hPa / inHg / Pa)OpenWeatherforecast[i].main.grnd_level (hPa) — TRUE absolute surface pressure at venue elevation. NOT sea-level adjusted. inHg = hPa/33.8639. Pa = hPa×100.
Sea-Level Ref / MSLPCalculatedHypsometric correction of station pressure up to sea level. Display only — never used for density.
MSLP = P_station × exp(λ×h / T)  |  λ=0.0065 K/m
Air Density (kg/m³)CalculatedMoist air density from station pressure, temp, humidity. pressure_calc.moist_air_density()
ρ = (P_dry / R_d×T) + (P_vap / R_v×T)  |  R_d=287.05, R_v=461.495
% of ISA (sea-level)Calculated ISA ModelActual density vs ISA sea-level standard (1.225 kg/m³).
ρ_actual / 1.225 × 100
% of ISA (@ elevation)Calculated ISA ModelActual density vs what ISA predicts at this venue's elevation.
ρ_actual / ρ_ISA_elev × 100
ISA Reference (pressure/density/temp)ISA ModelICAO standard atmosphere at venue elevation. pressure_calc.isa_at_elevation(elev_m)
q = 1 − 0.0065×h/288.15  |  P = 1013.25×q^5.25588  |  ρ = 1.225×q^4.25588
Density Altitude (ft)CalculatedPressure altitude corrected for virtual temperature (humidity-adjusted). Pilot standard.
PA = (1 − (P/1013.25)^0.190284) × 145366.45  |  DA = PA + 120×(T_v − 288.15)
Headwind / Crosswind (mph)Calculated Venue DBWind decomposed relative to stadium field orientation. wind_calc.wind_vs_stadium()
rel = (wind_dir − field_orient) mod 360  |  headwind = speed×cos(rel)  |  crosswind = speed×sin(rel)
Indoor/HVAC DensityCalculatedFor INDOOR/DOME venues: uses HVAC baseline 70°F / 45% RH with outdoor station pressure. pressure_calc.indoor_hvac_model()
Buildings are not sealed — indoor absolute pressure = outdoor station pressure. Only T and RH differ.
